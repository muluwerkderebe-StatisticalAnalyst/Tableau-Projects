# 🌍 Global Sales Data Visualization Project
Interactive Tableau Dashboard for Sales Performance Insights
This repository showcases my Global Sales Data Visualization Project, an end‑to‑end business intelligence solution built using Tableau Public.
The dashboard provides a comprehensive view of global sales performance, enabling users to explore trends, identify high‑value markets, and uncover actionable insights.
[Muluwerk Derebe-Tableau Project](https://public.tableau.com/app/profile/muluwerk.derebe/viz/GlobalsalesDataVisualizationProject/SalesDashboard?publish=yes)

# 📊 Project Overview
The goal of this project is to transform raw sales data into a clear, interactive, and decision‑ready dashboard.
It highlights key business metrics such as:
- Total revenue and profit performance
- Regional and country‑level sales distribution
- Product category and sub‑category trends
- Customer segmentation and order patterns
- Time‑series analysis of sales and profit
The dashboard is designed for business leaders, analysts, and stakeholders who need a quick yet deep understanding of global sales dynamics.

# 🚀 Key Features
## 1. Global Sales Overview
- High‑level KPIs: revenue, profit, quantity sold
- Interactive filters for region, category, and time
## 2. Geographic Sales Map
- Visualizes sales distribution across countries
- Highlights top‑performing and underperforming markets
## 3. Product Performance Analysis
- Category and sub‑category breakdown
- Profitability insights for strategic decision‑making
## 4. Time‑Series Trends
- Monthly and yearly sales patterns
- Seasonality and growth analysis
## 5. Customer Insights
- Customer segments and order behavior
- Contribution to total revenue
